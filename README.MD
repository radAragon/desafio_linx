# Linx challenge

# Part-1

A simple API with a Product endpoint that receives POST requests to create and update products but rejecting repetitive payloads (double sends).

## Requirements:

* Docker
* Node 8+
* Run: `$ npm install`

## Usage:

* Run: `$ npm start`
* Send POST requests to: `localhost:3000/products`

## Test:

* Run: `$ npm test`
